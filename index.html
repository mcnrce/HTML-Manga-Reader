What's inside
Features
📖
5 Reading Modes
Long strip, Long strip with gaps, Paged LTR, Paged RTL, Paged vertical — all with smooth swipe navigation.
🔄
Screen Rotation
Portrait, Landscape (90°), Upside down (180°). Implemented via CSS transform — no system permissions needed.
📦
Offline Reading
Download chapters to IndexedDB or a device folder. Read without internet. Export as ZIP.
🔔
Updates Feed
Checks your library for new chapters. Snapshot-based comparison — only shows truly new chapters.
🎨
Custom Filter
Brightness, contrast, saturation, hue shift. Presets: Grayscale, Inverted, Warm, Cool.
🔌
Source System
Add any manga site as a source file. Sources run sandboxed — no access to your data or browser.
✅
Chapter Tracking
Auto-marks chapters read on last page. Unread badge on library covers. Bulk mark read/unread.
🗂
Library & Categories
Organize manga into custom categories. Sort, filter, and display options. Adaptive grid layout.
📱
Responsive
Fluid layout adapts from small phones to tablets. Navigation rail on wide screens. Works on Kiwi, Via, Firefox.

Write a Source
<!-- Place next to hub.html, then add from Browse → + -->
<script>

async function getPopular(p) {
  const data = await fetch('https://yoursite.com/api/popular?page='+p.page).then(r=>r.json());
  return {
    items: data.results.map(m => ({ id: m.id, title: m.title, cover: m.thumb })),
    hasNext: data.hasNextPage
  };
}

// Also implement: getLatest, search, getDetails, getChapters, getPages

window.__SOURCE__ = {
  MANIFEST: { id: 'mysource', name: 'My Source', icon: '📖', version: '1.0', langs: ['en'] },
  getPopular, getLatest, search, getDetails, getChapters, getPages
};
</script>
